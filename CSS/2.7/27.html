Uppgift 2.7 Sidlayout

På denna sida kommer vi gå igenom hur man positionerar sina element på sidan. Man bör använda CSS framför tabeller då tabeller huvudsakligen är väldigt svåra att stilsätta i jämnförelse med ett DIV-elemnet. Det är även betydligt svårare att ställa in exakta storlekar då en tabell hänger ihop.

  • position - Detta attribut gör det möjligt att bestämma hur ett objekt skall tolkas av sin omgivning. Ett objekt med position: absolute påverkar/påverkas inte av andra element och kan positioneras fritt över sidan.
  • top - Bestämmer avståndet mellan objektets förälder och sig själv, utifrån toppen av objektet.
  • left - Bestämmer avståndet mellan objektets förälder och sig själv, utifrån vänster av objektets kant.
  • bottom - Bestämmer avståndet mellan objektets förälder och sig själv, utifrån botten av objektets kant.
  • right - Bestämmer avståndet mellan objektets förälder och sig själv, utifrån höger av objektets kant.
  • z-index - Bestämmer i vilken ordning ett objekt skall ligga skikt mässigt. Om ett element har z-index: 1; och dess granne har z-index: 2; och båda har en brädd på 100% så kommer det objektet med z-index: 2; ligga övers för den har ett högre prioritets värde.
  • float - gör det möjligt att positionera ett element åt höger eller vänster. Detta gör det möjligt att positionera element intill varanadra horisontellt.
  • clear - Gör det möjligt att ställa in så man inte påverkas av flytande objekt från bestämmt håll.
  • vertical-align - Gör det möjligt att ställa in var ett element skall positioneras i förhållande till föräldern.
  • Exempel: Fixerad, Flytande, Elastisk

Denna sektion går igenom hur man sätter storlekar på sina element

  • width - Bestämmer bredden på ett element, med godtycklig enhet.
  • height - Bestämmer höjden på ett element, med godtycklig enhet.
  • max-width - Gör det möjligt att bestämma max-bredden på ett element i godtycklig enhet
  • max-height - Gör det möjligt att bestämma max-höjd på ett element i godtycklig enhet
  • min-width - Gör det möjligt att bestämma minimum-bredden på ett element i godtycklig enhet
  • min-height - Gör det möjligt att bestämma minimum-höjd på ett element i godtycklig enhet
